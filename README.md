# Image Compressor

Minimal web app to compress images instantly.

No signup.  
No API.  
Just clean compression.

---

## 🚀 Live Demo



## ✨ Features

- Compress images to **WebP / JPG / PNG**
- Import **HEIC** photos from iPhone (auto converts to JPG for processing)
- Before/After **compare slider**
- Optional resize with **lock ratio**
- Drag & drop support
- Download compressed images instantly
- No authentication
- No ads
- No watermarks

---

## 🖥️ How It Works

1. Upload an image (or drag & drop)
2. Everything is processed fully in the browser
3. Pick output format + quality (and resize if needed)
4. Compress and compare results instantly
5. Download the compressed image
